Apache configuration

Note that ROOT_HTTP_CORE and ROOT_HTTP_MODULES starts with a "/"! (Windows) Note that path-seperators have to be "/" in ROOT_HDD_CORE and ROOT_HDD_MODULES!

Module activation

LoadModule alias_module modules/mod_alias.so LoadModule rewrite_module modules/mod_rewrite.so

Alias

Alias ROOT_HTTP_MODULES/ "ROOT_HDD_MODULES/"
<Directory "ROOT_HDD_MODULES">
    #Allow from all
    Require all granted
    AllowOverride All
</Directory>

# https://httpd.apache.org/docs/2.4/upgrading.html # 2.2 configuration: # Order allow,deny # Allow from all # 2.4 configuration: # Require all granted Alias ROOT_HTTP_CORE/ "ROOT_HDD_CORE/"
<Directory "ROOT_HDD_CORE">
    #Allow from all
    Require all granted
    AllowOverride All
</Directory>

Rewrite

RewriteEngine on

Rewrite for individual extension

RewriteRule ^ROOT_HTTP_MODULES/(.*)\.EXTENSION$ "ROOT_HDD_MODULES/$1.php" RewriteRule ^ROOT_HTTP_CORE/(.*)\.EXTENSION$ "ROOT_HDD_CORE/$1.php"
