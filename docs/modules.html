Modules

Naming

A module has two names: A cleartext title and a short stringId. The namespace for the stringId is very restricted. The name occures in: * PHP variable names * DB table names * Windows/Unix folder names Additionnaly it will be pre- and suffixed (by underscore) in several contexts. The remaining available characters are:

[A-Za-z0-9]

The lowercase stringId of the module has to be unique in each project To ensure uniqueness the module-IDs have to be renamed possibly on import.
