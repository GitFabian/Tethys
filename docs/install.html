Bestehendes Projekt, Tethys als Submodul

1.) Projekt versionieren Kein bestehendes Repository? Projektordner: git init git add . git commit -m "Initial commit" 2.) Tethys-Repository als Submodul anlegen Projektordner: git submodule add https://github.com/GitFabian/Tethys.git tethys [ DEBUGGING ] 3.) Submodul-Eintrag committen Projektordner: git commit -m "Add Tethys as a submodule"

Bestehendes Projekt mit Submodul klonen

Quelle: https://git-scm.com/book/en/v2/Git-Tools-Submodules 1.) Projekt klonen Projektordner: git clone REPO_URL . 2.) Submodule klonen Projektordner: git submodule init git submodule update -ODER:- 1.) In einem Abwasch: Projektordner: git clone --recurse-submodules REPO_URL .
