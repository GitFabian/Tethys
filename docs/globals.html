Constants

EXTENSION

Virtual extension for your php files. Must be configured in Tethys and in Apache. Default value: php Example: html

ROOT_HDD_CORE

Root path to Tethys when accessed via file system. Examples: C:\Users\Foo\Workspace\Tethys C:/Users/Foo/Workspace/Tethys /var/www/project1/tethys In windows paths "\" has to be converted to "/" at some points!

ROOT_HTTP_CORE

Root path to Tethys when accessed via browser. Default value: /tethys
